# Expense Tracker

A web application for managing and tracking daily expenses.

## Features
- Add expenses
- Edit expenses
- Delete expenses
- Search by title or category
- Calculate total spending
- Bootstrap UI

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- EJS
- Bootstrap

## Run Locally

```bash
npm install
nodemon app.js
```
